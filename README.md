# Google Translate Extension

基于 [Google 翻译(cn)](https://translate.google.cn)，无需科学上网，无需 API Key 的翻译扩展。

## Features

包含以下功能：

1. 自动中英互译；
2. 复制翻译结果到剪切板；
3. 候选词选择；
